### 🎬 一、 视觉动效重构：打造真正的“电影感”

我们要彻底改变“死板的文字”和“无聊的切换”，引入**“呼吸感”**。

**1. 图片无损微动（Ken Burns 2.0）：**

* **背景模糊层：** 提取当前照片，放大 150% 并加上极强的高斯模糊作为全屏背景，并且**让这个模糊背景缓慢地向左或向右漂移**。
* **前景主体层：** 照片本身 100% 完整显示（绝不裁剪）。并且让这张照片极其缓慢地**从 1.0 放大到 1.05 倍**。这样既有动态的纵深感，又绝对不会切掉表妹的脸。

**2. 字幕级文字排版（电影感排版）：**

* **位置下移：** 绝对不能挡住画面正中心！把文字移动到屏幕**下方 15% 的黄金分割点**（就像电影字幕一样）。
* **高级特效：** 文字出场不能生硬，要用**“诗意浮现（Fade & Blur-out）”**——字从轻微模糊变得清晰，并在图片切换前慢慢像烟雾一样消散。
* **字体设计：** 选用细体字（或极简宋体），加上淡淡的黑色发光阴影（Drop Shadow），保证在任何颜色的照片上都清晰可见，极具文艺气息。

### 📖 二、 100+ 张照片的文案策略：【留白与随机】

如果你有 100 张照片，**千万不要每张照片都配一句文案！**
这就像看电影每一秒都在说话，观众会视觉疲劳的。真正的高级感在于**“留白（节奏感）”**。

**💡 设计逻辑：**

1. 建立一个包含 20-30 句精美祝福语的“文案库”。
2. 让代码**随机但稀疏**地抽取文案：比如大约 **每放 3~4 张照片，才浮现出一句走心的文案**。
3. **无文案的照片显示什么？** 优雅地显示一个时间戳、或者一句极简的英文，比如 `「 CHAPTER 22 」` 或 `「 MEMORIES 」`。

这样，随着音乐的起伏，她看着自己的一张张照片流转，偶尔浮现一句戳心的话，这种“不经意的温柔”杀伤力是最大的！

# Task: Overhaul the final "Cinematic Photo Reel" (Slideshow) for a 100+ images gallery.

The user was unhappy with the previous slideshow. The text was blocking the center of the images, the images were static (no zoom), and loading 100 photos was not handled well. 
Please rewrite the photo reel logic (`openReel` or equivalent in `app.js`) with the following strict requirements:

## 1. Cinematic Visuals & No Cropping
* **Background (Ambient Blur):** For every slide, create a full-screen background using the SAME image, but apply `filter: blur(40px) brightness(0.4)`. Animate this background with a very slow horizontal CSS pan (`translateX`).
* **Foreground Image:** The actual photo MUST use `object-fit: contain` so it is NEVER cropped.

* **Animation:** The text should fade in with a slight blur reduction (`filter: blur(10px)` to `blur(0)`), stay for a few seconds, and fade out before the image changes.
