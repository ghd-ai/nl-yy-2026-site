### 🕵️‍♂️ 第四关终极完整设计案：【模仿犯的密信与法医滤镜】

#### 1. 深度致敬（解决问题 1：影视剧元素的硬核融合）

* **致敬《模仿犯》的“挑衅信”：** 连环杀手（这里化名为“时间窃贼”）留下了一封 6x6 的字母乱码信，这是典型的高智商犯罪挑衅（类似著名的“十二宫杀手密码”）。
* **致敬《法医秦明》的“多光谱检视台”：** 面对肉眼无法辨认的乱码，法医不会去盲目瞎猜。秦明的做法是使用物理手段——**“法医多光谱检视台”**。你需要将收集到的 3 张嫌疑人活动轨迹（胶片），作为“光学遮罩滤镜”叠在信件上，过滤掉杂讯，提取真相。

#### 2. 幽默解构（解决问题 2：拒绝说教，搞笑吐槽）

我们把线索写成一本正经的法医报告，但内容全是对她日常的**“搞笑吐槽”**。每一句吐槽里，都精准地藏着胶片的【放置位置】和【旋转角度】。

* **胶片 A【志愿活动轨迹】** -> 目标坐标：`不旋转(0度) + 右下角`
* *法医行为侧写：* “据监测，该探员每次做志愿活动，微信步数都能走到朋友圈**最底层（垫底）**；但只要一听见活动结束去干饭，必定立刻向**右**转百米冲刺，且**从不回头（0度不变）**。”


* **胶片 B【正大杯调研波形】** -> 目标坐标：`旋转180度 + 正上方`
* *法医行为侧写：* “因为正大杯问卷改了 **180 遍**，导致该探员精神状态时常发生**180度大翻转**。但她放出狠话：就算头秃，这个杯我也必须给它端到**最上面（顶端）**！”


* **胶片 C【悬疑剧心率切片】** -> 目标坐标：`旋转90度 + 正中央`
* *法医行为侧写：* “半夜看悬疑剧，常被剧情吓得在被窝里 **90度** 鲤鱼打挺。但那颗永远八卦且热爱推理的**心**，始终坚定地放在**正中间**。”



*（这种文案她一看就会笑骂一句“有病吧”，但马上又能反应过来：哦，最底层+右转 = 放在右下角；180度 = 双击旋转两次！）*

#### 3. 视觉惊艳双重奏（解决问题 3：不仅仅是字母）

如果拼出来只是 C-A-K-E 四个字母，确实太简陋了。我们将引入**“视觉欺骗与动画反转”**。

* **第一层（滤网解密）：** 当 3 张胶片按正确位置叠好时，乱码被黑色遮住，透明的窟窿里露出了 **C、A、K、E** 四个发光的字母。
* **第二层（华丽变身）：** 当她输入 CAKE 点击通关的瞬间，**“法医检视台”的白光会突然变成温暖的暖黄色**。
* 那 3 张原本黑乎乎的胶片，在暖光下突然褪去了伪装——它们的黑色边框竟然拼成了一个**巨大的、三层高的卡通生日蛋糕！**（C、A、K、E 四个字母刚好变成了蛋糕上的四根蜡烛）。
* *（这在前端实现非常简单，通关瞬间切换胶片的 CSS 背景图即可。）*
