# 🎂 第一关设计案：消失的许愿光

## 1. 核心概念

* **案件名：** 《月月顶牛庆生夜·消失的许愿光》
* **侦探：** **月月侦探 (Detective Yue)**
* **案情：** 距离零点许愿只剩 30 分钟，传说中能实现愿望的**“月光烛台”**不见了。现场的五位亲友（嫌疑人）都在各自忙碌，你需要通过他们的证词，找出烛台在谁手里。
* **风格：** 综艺《明星大侦探》风格（花字、音效、搜证板），但充满家庭聚会的温馨感。

## 3. 逻辑谜题 (The Puzzle)

**通关条件：** 侦探需要将底部的**道具图标**和**位置图标**，拖拽到正确的**角色卡片**上。

**七条关键线索 (搜证板内容)：**

1. **【东代码的不在场证明】**：作为程序猿，**东代码**正如他在群里说的那样，把自己关在**书房**里敲**笔记本电脑**。
2. **【冬贪吃的行踪】**：那个**拿着草莓蛋糕**的小家伙（冬贪吃），为了怕被骂，躲到了**阳台**去偷吃。
3. **【辰统筹的焦虑】**：**辰统筹**嫌客厅太吵，躲在**卧室**里反复核对**流程手卡**。
4. **【菲镜头的站位】**：拿着**单反相机**的人（菲镜头），正站在**东代码**（书房）和**辰统筹**（卧室）中间的**走廊**里找角度。
5. **【关键证词 A】**：**俊魔术**既不在阳台，手里拿的也不是蛋糕。
6. **【关键证词 B】**：**客厅**里的人，并没有拿流程手卡。
7. **【真相拼图】**：大家都在找的**月光烛台**，其实就在**客厅**里。

**(逻辑闭环：只有“俊魔术”在“客厅”，手里拿着“月光烛台”。)**

## 4. 视觉与交互细节

* **开场：** 模仿明侦的自我介绍。点击角色头像，会弹出一句语音气泡（比如冬贪吃说：“蛋糕真甜！”）。
* **投票机制：** 界面右下角有一个**金色的鸟笼**。
* **错误反馈：** 如果拖错位置，屏幕震动，弹出弹幕：“侦探，这个推理有点离谱哦~”
* **成功反馈：**
1. 选中“俊魔术”点击检举。
2. **“铁笼”落下**，关住俊魔术。
3. 俊魔术变出一朵玫瑰花，铁笼瞬间变成礼花炸开。
4. 烛台点亮，背景音乐切换为温馨版。
