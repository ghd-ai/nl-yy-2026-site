移动端你这个项目最常见的“痛点”，基本就两类：

1) **顶部 sticky 栏太高**（你现在在 `max-width:620px` 直接把 topbar 变成两行/多行，空间被吃掉）
2) **iPhone 安全区 + 底部固定按钮/Toast/弹层**（容易被刘海/小横条挡住，或显得拥挤）

下面给你一套“改动很小、效果立竿见影”的修法；然后再给你**完全静态托管**（不用你电脑开后端，别人直接访问）的部署方案。

---

## 一、移动端怎么修（推荐先做这 3 个补丁）

### 补丁 1：让 topbar 在手机上变“瘦”，别再纵向堆叠
你现在 `styles.css` 里这段会让 topbar 在手机变很高：

```css
@media (max-width: 620px) {
  .topbar { flex-direction: column; ... }
}
```

把这段 **整块替换**成下面这一块（核心：手机端仍然横向排列、隐藏 subtitle、缩小 padding）：

```css
@media (max-width: 620px) {
  .app {
    padding: 12px 12px 42px;
  }

  .topbar {
    flex-direction: row;         /* 不再竖着堆 */
    align-items: center;
    justify-content: space-between;
    flex-wrap: wrap;             /* 真放不下就换行，但不会像原来那么高 */
    gap: 8px;
    padding: 10px 10px;
    top: calc(8px + env(safe-area-inset-top));
  }

  .brand-subtitle {
    display: none;               /* 手机隐藏副标题，立刻省一行高度 */
  }

  .brand-title {
    font-size: 14px;
  }

  .topbar-right {
    gap: 8px;
    justify-content: flex-end;
  }

  .score-pill {
    padding: 6px 10px;
  }

  .topbar-actions {
    gap: 6px;
  }

  .topbar-actions .btn {
    padding: 8px 10px;
    border-radius: 10px;
    font-size: 12px;
  }

  .grid.cols-2,
  .grid.cols-3 {
    grid-template-columns: 1fr;
  }
}
```

如果你想更“狠”一点（手机上直接不要 sticky），再加一行即可：
```css
@media (max-width: 620px) { .topbar { position: static; } }
```

### 补丁 2：把所有“固定在底部/顶部”的东西让出安全区
加一段通用的 safe-area 变量（放在 `:root` 下面就行）：

```css
:root{
  --safe-top: env(safe-area-inset-top);
  --safe-bottom: env(safe-area-inset-bottom);
  --safe-left: env(safe-area-inset-left);
  --safe-right: env(safe-area-inset-right);
}
```

然后加这些覆盖（不用删原来的，后面追加即可）：

```css
/* Toast 不要压住 iPhone 小横条 */
.toast {
  bottom: calc(20px + var(--safe-bottom));
}

/* 弹层整体 padding 给安全区 */
.overlay {
  padding: calc(18px + var(--safe-top)) calc(18px + var(--safe-right))
           calc(18px + var(--safe-bottom)) calc(18px + var(--safe-left));
}

/* 右上角 theater-hud 给安全区 */
.theater-hud {
  top: calc(18px + var(--safe-top));
  right: calc(18px + var(--safe-right));
}

/* 右下角“点击跳过”等按钮给安全区 */
.cinema__skip {
  bottom: calc(18px + var(--safe-bottom));
  right: calc(18px + var(--safe-right));
}

/* Case2 那个底部卡片也给安全区（你有用 fixed bottom:20px） */
.mp-message-card {
  bottom: calc(20px + var(--safe-bottom));
}
```

你 case1 的 `cage-btn` 在手机端有一个 `bottom: 18px;` 的 media query，把它改成：
```css
@media (max-width: 620px) {
  .cage-btn { bottom: calc(18px + var(--safe-bottom)); }
}
```

---

### 补丁 3：修一下移动端 100vh 的“地址栏跳动”
把你现在的：

```css
body { min-height: 100vh; }
```

改成（保留 100vh 作为 fallback）：

```css
body {
  min-height: 100vh;
  min-height: 100dvh;
}
```

这能减少移动端浏览器地址栏展开/收起导致的高度抖动。
